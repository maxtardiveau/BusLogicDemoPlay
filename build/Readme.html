Bus Logic Demo (Play edition)

What is this?

This directory contains a copy of ABL's BusLogicDemo project, pre-configured for the Play framework.
This is a simple Play application, demonstrating the use of Automated Business Logic's transaction logic engine.

Running the demo app

This demo can either be run from the command line, or from an IDE. But regardless, you will need Play 1.2 installed and running.

You will also need to update the dependencies. Change your current directory to BusLogicDemoPlay and execute:

play dependencies --sync

From the command line

To run this demo from the command line, make sure that Play is set up correctly. Change your current directory to BusLogicDemoPlay and execute:
play run

This may take a minute, as Play will download the required libraries.

From an IDE

Before you import this project into your IDE, you will need to ask Play to generate the appropriate project files. To do so, you should change your current directory to BusLogicDemoPlay and execute:
  • play eclipsify for Eclipse
  • play idealize for IntelliJ
  • play netbeansify for NetBeans
You can then import the project into your IDE.

To open this demo project in Eclipse or STS, select File/Import/Existing Project into Workspace and select the BusLogicDemoPlay directory. IntelliJ and NetBeans have a slightly different procedure, but they seem to be able to open the project with little difficulty.

If you're not sure how to start the application, please refer to the Play documentation.

More information about this demo is available on Automated Business Logic's web site.
